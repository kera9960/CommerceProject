# 프로젝트 소개

## 주요 기능
    카테고리 목록 조회
    상품 목록 조회
    상품 상세 확인
    장바구니 담기
    장바구니 조회
    주문 확정
    주문 취소
    재고 확인 및 차감

## 클래스 구조
    Main : 프로그램 실행을 시작하고 CommerceSystem을 실행
    Product : 상품명, 가격, 설명, 재고를 가짐
    Category : 카테고리 이름과 상품 목록을 관리
    Customer : 고객명, 이메일, 등급 등 고객 정보를 저장
    CartItem : 장바구니에 담긴 상품과 수량 정보를 저장
    Cart : 장바구니 목록과 총액 계산 등을 담당
    CommerceSystem : 전체 실행 흐름과 사용자 메뉴를 제어
    InputParser : 문자열 입력을 숫자로 변환하고 입력값은 검증
    ProcessOrder : 주문 확정 및 취소 처리 담당

## 실행 방법
    Main 클래스 실행
    콘솔에서 메뉴 번호 입력
    카테고리 선택 후 상품 조회
    상품을 장바구니에 담고 주문 진행
